# 💤 Luki's Neovim configution

Basic Neovim configuration based on [LazyVim](https://github.com/LazyVim/LazyVim).

## Installation

### Linux/WSL/MacOS

```bash
mv ~/.config/nvim{,.bak}

git clone https://github.com/luki446/nvim-config ~/.config/nvim
```

### Windows + Powershell

```powershell
Move-Item $env:LOCALAPPDATA\nvim $env:LOCALAPPDATA\nvim.bak

git clone https://github.com/luki446/nvim-config $env:LOCALAPPDATA\nvim
```
